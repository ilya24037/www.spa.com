# ФИНАЛЬНОЕ РЕШЕНИЕ: Исправление дублирования черновиков

## Проблема решена!

Данные от backend приходили вложенными в объект 'data' из-за использования AdResource.

### Решение:
В Edit.vue теперь корректно извлекаются данные из вложенной структуры:
- ad.data?.id вместо ad.id
- ad.data вместо ad как initialData

### Результат:
- ID корректно передается в AdForm
- При сохранении отправляется PUT /draft/{id}
- Дубликаты больше не создаются

Проверено и работает!
