Для активных объявлений у администратора должен быть
  следующий функционал:

  Функционал администратора для активных объявлений:

  1. Просмотр и мониторинг

  - ✅ Видеть все активные объявления всех пользователей
   (уже реализовано)
  - Просматривать детальную информацию о каждом
  объявлении
  - Видеть статистику просмотров, звонков, добавлений в
  избранное

  2. Модерация и контроль

  - Заблокировать объявление - если обнаружено нарушение
   правил
  - Снять с публикации - временно скрыть объявление
  - Редактировать - исправить ошибки/нарушения в тексте
  или фото
  - Добавить пометку модератора - внутренние заметки о
  проблемах

  3. Управление статусом

  - Отправить на перемодерацию - если нужна повторная
  проверка
  - Перевести в архив - для неактуальных объявлений
  - Продлить размещение - если у пользователя истекает
  срок

  4. Действия поддержки

  - Поднять в поиске - помочь пользователю с
  продвижением
  - Сбросить счетчики - обнулить статистику при
  необходимости
  - Изменить тариф - перевести на платный/бесплатный

  5. Работа с жалобами

  - Просмотреть жалобы на конкретное объявление
  - Принять решение по жалобе
  - Уведомить пользователя о принятых мерах

  На скриншоте видно, что сейчас есть:
  - ✅ "Поднять просмотры"
  - ✅ "Снять с публикации" (в выпадающем меню)

  Нужно добавить:
  - "Заблокировать"
  - "Редактировать"
  - "Отправить на перемодерацию"
  - "Просмотр жалоб"

  Это стандартный набор для маркетплейсов типа Avito.
