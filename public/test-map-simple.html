Простая карта без Vue

Тестирование Яндекс.Карт API напрямую без фреймворка

Готов к поиску. Введите адрес и нажмите "Найти" или кликните на карту.

Информация:

  • Кликните на карту для установки маркера
  • Перетащите маркер для изменения позиции
  • Используйте поиск для быстрого перехода к адресу
  • Координаты: не выбраны
